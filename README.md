# AI Squat Coach

This project uses the pretrained YOLOv8 pose model from Ultralytics to
estimate human pose and score squat posture based on joint angles.

## Environment

- Python 3.12
- PyTorch 2.9.1
- torchvision 0.24.1
- ultralytics 8.3.229
- opencv-python 4.12.0.88
- numpy 2.2.6

Install dependencies:

pip install -r requirements.txt
